# Egg Timer
A simple and interactive Egg timer that helps you to cook eggs to perfection.
Choose the egg style,start the timer and get the exact doneness you want 

## Features
- Multiple egg cooking presets
- Simple timer interface
- Visual egg indicators for each doneness level
- Easy to use and beginner friendly
- Perfect for quick kitchen timing

## How to Use
- Select the type of egg you want to cook
- Press Start timer
- Wait for the timer to finish
- Remove the egg and enjoy
